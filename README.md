# Vox — Vocal Toning

A vocal toning companion that runs entirely in the browser. Tone the seven
chakras with a shruti-box drone — a full root-to-crown journey or one centre
at a time — while a live listening engine measures pitch, breath, clarity,
and resonance and turns them into a single coherence score.

Everything is analysed locally in real time. Your voice is never recorded,
stored, or uploaded.

## Hosting

This is a single static page (`index.html`) with zero dependencies.
GitHub Pages serves it directly: **Settings → Pages → Deploy from a branch →
`main` / `(root)`**.
